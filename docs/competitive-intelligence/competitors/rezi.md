# Rezi

**Last verified date:** 2025-12-27

## Overview

| Field | Value |
|:------|:------|
| **Product** | Rezi AI Resume Builder |
| **URL** | https://rezi.ai |
| **Category** | AI Resume Builder / ATS Optimization |
| **Target user** | Job seekers who want to build ATS-optimized resumes from scratch |

## Positioning Promise

"The leading AI resume builder" — Rezi positions as an AI-powered resume creation tool that guarantees ATS compatibility with real-time content analysis and keyword targeting.

## Core Workflow Summary

1. Start from scratch or import existing resume
2. AI generates bullet points based on job title/role
3. Real-time "Rezi Score" provides instant feedback
4. Keyword targeting suggests missing terms
5. Download ATS-optimized PDF

## Feature Highlights

- **AI Resume Writer**: GPT-powered bullet point and summary generation
- **Rezi Score**: 23-criteria evaluation with detailed breakdown
- **Keyword Targeting**: AI-identified skills and terms to include
- **Real-Time Content Analysis**: Live feedback as you type
- **AI Bullet Point Editor**: Rewrite suggestions for existing content
- **AI Summary Generator**: One-click professional summary
- **Cover Letter Generator**: Matched drafts from resume + JD
- **Resignation Letter Generator**: Bonus utility tool

## Pricing and Packaging

| Plan | Price | Notes |
|:-----|:------|:------|
| **Free** | $0 | 1 resume, limited AI, 3 PDF downloads |
| **Basic** | $3/mo | AI writing, keyword targeting, analysis |
| **Pro** | $29/mo | Unlimited AI, resumes, downloads, monthly review |
| **Lifetime** | $129-149 one-time | All Pro features permanently |

**Conversion mechanics:**
- Low-cost entry ($3/mo) reduces commitment anxiety
- Lifetime deal creates urgency with limited availability
- 30-day money-back guarantee lowers risk
- Discount codes frequently available (20-40% off)

## Output Quality and Trust Signals

- **23-criteria Rezi Score** provides detailed feedback breakdown
- **ATS-friendly templates** explicitly marketed
- **Real-time analysis** shows active improvement tracking
- **No explicit recruiter simulation** (weakness)

## Notable Strengths

1. **Aggressive pricing** with $3/mo entry and $129 lifetime options
2. **Comprehensive AI tooling** for content generation
3. **Real-time feedback** creates "live" improvement experience
4. **Lifetime deal** appeals to budget-conscious users
5. **Clean, modern interface** feels professional

## Notable Weaknesses

1. **Builder-focused** — less useful for reviewing existing resumes
2. **Rigid templates** compared to design-first alternatives
3. **No recruiter perspective** — ATS-only simulation
4. **No LinkedIn optimization** (Resume Worded gap)
5. **No Chrome extension** or job tracking

## Why Users Choose Them

- Lifetime deal is compelling value proposition
- AI generation speeds up resume creation
- Real-time feedback feels interactive
- ATS optimization provides confidence

## What We Should Steal

- **Aggressive entry pricing**: $3/mo reduces friction
- **Lifetime deal concept**: One-time payment appeals to job seekers
- **Real-time feedback**: Live scoring creates engagement
- **Discount codes**: Promotional pricing drives conversion

## Where We Can Win

1. **Review vs build**: We analyze existing resumes, not just create new
2. **Recruiter simulation**: Human-like judgment vs bot optimization
3. **Editorial authority**: "Principal Recruiter" voice vs AI generation
4. **Interactive rewrites**: "Red Pen" corrections vs static suggestions
5. **Premium UX**: Fraunces typography, glassmorphism vs standard SaaS
