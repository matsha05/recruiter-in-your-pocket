## Core Workflow Summary

1. Create/import resume (LinkedIn import supported)
2. Clip jobs from boards via Chrome extension
3. Tailor resume to each job description
4. Track applications in Kanban-style board
5. Monitor progress with insights dashboard

## Feature Highlights

- **Job Application Tracker**: Kanban board for application status management
- **Chrome Extension**: One-click job clipping from LinkedIn, Indeed, etc.
- **AI Resume Builder**: Templates with AI-generated bullet points
- **Match Score**: Percentage-based resume-to-JD matching
- **Keyword Matcher**: Checklist format for missing keywords
- **Cover Letter Generator**: AI-assisted drafts matched to JD
- **Design Mode (Premium)**: Full visual customization controls
- **Skills Generator**: Extracts critical skills from job descriptions

## Notable Strengths

1. **Comprehensive platform**: Resume + Tracking + Applications in one
2. **Chrome extension** creates daily touchpoint with users
3. **Generous free tier** removes friction for adoption
4. **Modern UI/UX**: Clean, professional, not overwhelming
5. **LinkedIn import** simplifies onboarding
6. **Active community** and responsive development

## Notable Weaknesses

1. **Feature complexity** can feel like admin work
2. **Template designs** perceived as plain/limited
3. **AI content accuracy** requires manual editing
4. **No dedicated mobile app**
5. **Learning curve** for full platform utilization
6. **Subscription model** still creates fatigue for episodic users

## Why Users Choose Them

- "I need one tool for everything" appeal
- Free tier is legitimately useful
- Chrome extension makes job clipping frictionless

## What We Should Steal

- **Chrome extension strategy**: Process-layer presence across job boards
- **Generous free tier**: Lower barrier drives adoption
- **Version management**: Track resume iterations easily

## Where We Can Win

1. **Focus vs sprawl**: Surgical resume review vs feature-heavy CRM
2. **Price model**: Credit-based ($9 one-time) vs subscription
3. **Editorial depth**: Expert verdicts vs checklist optimization
4. **Recruiter authority**: "What they see" vs "what bots parse"
